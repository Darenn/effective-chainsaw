Detailed Description

template<class Element>
class Heap_Id< Element >

This class implements a generic heap with id for the elements.

It uses a binary tree such that the value held in any node is greater (or equal) to the value in its sons.

Auxiliary arrays are used to go from id to positions and to record available id.

Precondition
Element must be comparable: operators < and <= must be defined.

Implementation:

  • the tree is folded into an array.
  • reference / pointers are used to store elements (i.e. no copy is made)
